## Design System

### Core Values
- **Bold**: Confident, direct, no-nonsense
- **Simple**: Clean interface, easy to understand
- **Geometric**: Sharp edges, squares, blocks
- **Energetic**: Vibrant colors, dynamic

### Visual Style
- **Design Language**: Neubrutalism
- **Key Elements**:
  - 3px black borders
  - Flat vibrant colors
  - Sharp corners (no border radius)
  - 4px hard shadows
  - Bold typography (800-900 weight)
  - High contrast

### Color Palette

**Primary Colors**:
- Yellow: `#FFD700` - Main brand color
- Pink: `#FF69B4` - Secondary accent
- Cyan: `#00FFFF` - Tertiary accent
- Lime: `#00FF00` - Success/completion
- Orange: `#FF6B35` - Warning/energy

**Neutrals**:
- Black: `#000000` - Borders, text
- White: `#FFFFFF` - Backgrounds
- Light Gray: `#F5F5F5` - App background

### Typography

**Primary Font**: System Bold
- **Headings**: 900 weight, 32-48px
- **Titles**: 800 weight, 24-28px
- **Body**: 600-700 weight, 16-18px
- **Small**: 600 weight, 14px

**Principles**:
- Always bold or bolder
- No thin weights
- High contrast with background
- Readable at small sizes
